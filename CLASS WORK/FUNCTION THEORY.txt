Function : It is a piece of code, which performs a specific task.
		   
syntax : 	data-type function_name()
			{
				//statements
			}
			
Types of Function. (2 types)

1) Predefined Type.
	ex. printf(),scanf()
	
2) Userdefined Type.

	e.g void main()
		{
			//statement;
		}

         Categories of Functions.

Category 1 :

A function with no return type and with no parameter.

e.g void f1()
	{
		//statement.
	}

Category 2 :

A function with no return-type and with Parameter.

ex- void f1(int a)
	{
		//statement;
	}

Category 3 :

A function with return-type and with Parameter.

e.g int f1(int a,int b)
	{
		//code
	}

3 facts about the function.

1) Function Create/Defined.
2) Function Use.
3) Function Declaration (optional) 
		- if you create function before main().
